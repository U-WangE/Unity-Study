
# Unity LifeCycle

Unity의 기본적인 생명주기(LifeCycle)
-> Unity 내에 돌아가는 함수의 순서, 시기 등을 적어놓았다.
>
```javascript
private void Awake();
private void OnEnable();
private void Start();
private void FixedUpdate();
private void Update();
private void LateUpdate();
private void OnDisable();
private void OnDestroy();
```
설명하고 있는 함수 입니다.
